# Morpheme Tokenizer

An elegant, interactive, and responsive web application designed to deconstruct Russian words into morphemes (prefixes, roots, suffixes, endings).

## Features
- **Visual Breakdown:** Clearly distinguishes different morpheme types with color-coded tags.
- **Detailed Legends:** Learn the grammatical function of each identified morpheme.
- **Responsive Design:** Designed with Tailwind CSS to look stunning on both mobile and desktop screens.

## Quick Start
Open `index.html` in any browser to launch the interactive application.
